# 35-SOPKATHON-iOS-iOS2
35기 솝커톤 iOS 2팀 - iOS 레포입니다.

## 🌴 소개
서비스 이름: 정글 탈출

소개: 정글과 다름없는 거친 세상에서 살아남기 위해 퀘스트를 깨며 작은 성취감을 쌓아나가는 서비스입니다.

## 🌴 사용 기술
* Frameworks: UIKit, SwiftUI
* Libraries: Alamofire, Kingfisher, Moya, ReactiveSwift, RxSwift, SnapKit, Then
